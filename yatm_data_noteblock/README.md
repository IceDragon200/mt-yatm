# YATM Data Noteblock

A noteblock driven by the data network.

## Available Sounds

- Bass Drum (Synthesized)
- Snare Drum (Synthesized)
- HiHat (Synthesized)
- Rimshot (Synthesized)
- Melodic Drum (Synthesized)
- Bass Guitar (Synthesized)
- Guitar
- Piano
- Flute (Synthesized)
- Pulse (25% Duty) (Synthesized)
